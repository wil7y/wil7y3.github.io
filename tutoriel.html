1. Allumer une LED avec un bouton

Dans ce tutoriel, nous allons apprendre à allumer une LED en appuyant sur un bouton.

Code Arduino:


                const int pinLED = 9;
                const int pinBouton = 7;

                void setup(){
                    pinMode(pinLED, OUTPUT):
                    pinMode(pinBouton, INPUT_PULLUP);
                }

                void loop(){

                    int etatBouton = digitalRead(pinBouton);
                    if (etatBouton == LOW) {
                        digitalWrite(pinLED, HIGH);

                    } else {
                        digitalWrite(pinLED, LOW);
                    }
                }

2. Faire clignoter une LED automatiquement

Un projet simple pour faire clignoter une LED à un intervalle régulier.

Code Arduino:


                const int pinLED = 13;

                void setup(){
                    pinMode(pinLED, HIGH):
                }

                void loop(){

                    digitalWrite(pinLED, HIGH);
                    delay(500);
                    digitalWrite(pinLED, LOW);
                    delay(500);
                }
